# **練習卡：認識 Nomad Sculpt**

## 練習 001

![練習 1](images/1.png)

### 任務
通過修改基礎模型創建簡單的面部表情。

### 解決步驟
1. 使用「SelfMask」
2. 使用「Rect」選擇器遮罩眼睛部分
3. 使用「Move」工具改變嘴部
4. 清除遮罩
5. 使用「Smooth」工具去除凹凸不平的部分
